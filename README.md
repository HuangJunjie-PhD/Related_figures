# Related Figures

技术文章插图仓库

## 目录结构

- `pcie/` - PCIe 相关技术插图
  - `layer_architecture.png` - PCIe 三层架构
  - `tlp_structure.png` - TLP 包结构
  - `tlp_header_detail.png` - TLP 头部细节
  - `lcrc_mechanism.png` - LCRC 数据完整性保护
  - `sequence_number.png` - 序列号机制
  - `8b10b_encoding.png` - 8b/10b 编码
  - `128b130b_encoding.png` - 128b/130b 编码
  - `pam4_signaling.png` - PAM4 信号调制
  - `ltssm_state_machine.png` - LTSSM 链路训练状态机
  - 其他 PCIe 规范截图
